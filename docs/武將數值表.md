# 武將數值表

結構化資料見 [../data/generals.json](../data/generals.json)。傷害公式：`總傷害 = (基礎傷害 + 加法加成) × 乘法倍率`。

## 欄位說明

| 欄位 | 說明 |
|------|------|
| id | 唯一識別 |
| name | 顯示名 |
| faction | 陣營（魏/蜀/吳/群） |
| rarity | 稀有度（Common/Uncommon/Rare/Legendary） |
| skill_name | 技能名稱 |
| skill_trigger | 觸發時機 |
| skill_effects | 效果列表：type（加法/乘法）、value、condition |
| bonus_discards | （可選）額外棄牌次數加成 |

## 觸發時機

- **結算時**：雙方停牌後、傷害計算當下
- **要牌時**：玩家要牌且新牌入手後
- **停牌時**：玩家選擇停牌時
- **達成 21 點時**：玩家點數剛好 21 時
- **爆牌時**：玩家爆牌時
- **回合開始時**：發完初始兩張牌後
