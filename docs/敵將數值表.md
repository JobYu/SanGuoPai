# 敵將數值表

結構化資料見 [../data/enemies.json](../data/enemies.json)。敵將為關卡對戰之對手，與玩家方武將表分離。

## 欄位說明

| 欄位 | 說明 |
|------|------|
| id | 唯一識別 |
| name | 顯示名 |
| campaign | 所屬戰役（黃巾之亂 / 討伐董卓 等） |
| stage_index | 關卡序（1～11 對應前 11 關） |
| morale | 氣勢（生命值），歸零則敵將敗北 |
| ai_stand_threshold | 停牌閾值：敵將點數 ≥ 此值時傾向停牌 |
| ai_tendency | 性格標籤：莽撞 / 穩健 / 狡詐 |
| flavour | 一句話說明（劇情 / UI） |

## AI 行為對照（與 GDD 一致）

- **莽撞**：閾值 18～19，易貪牌。
- **穩健**：閾值 17，通常 17 點以下強制要牌。
- **狡詐**：閾值 15～16，較早停牌、偏保守。
